COMO FAZER DEGRADÊ EM CSS

para fazer um degradê em css abrimos a tag style no head e digitamos background-image, temos a opção de adicionar o gedrade linear (o usado normalmente) ou o radial circular, linea-gradient ou radial-gradient, o sentido adionamos to right para direita, to left para esquerda, to top para cima, to down para baixo e adicionamos quantas cores quisermos. Adicionamos background-attachment: fixed para o fundo ficar parado. e a tag global *{} e dentro dela height: 100% para ajustar o fundo e o degradê. para ajustar o angulo do desgradê usamos o valor e deg, exemplo 45deg
