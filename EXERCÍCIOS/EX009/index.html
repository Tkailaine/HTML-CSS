Listas ordenadas

a tag ol serve para listas ordenadas que importam a ordem e numeração dentro dela quando formos colocar um item na lista devemos por dentro da tag li, o parametro type no ol muda se vai ser numerada ou alfabetizada ou em romano

  1. acordar
  2. tomar banho
  3. escovar os dentes
  4. estudar

Lista não ordenadas

listas não ordenadas não importa a ordem. A tag usada é ul e para adicionar itens é a tag li, temos os type "disc, circle

  • feijão
  • arroz
  • cebola
  • alho
